# bookstore-goVersion
# 📚 Bookstore - 图书商城系统

一个基于 Go 语言开发的完整 Web 应用，适合刚学完 GoWeb 基础的同学们练手实践。

### 部署教程
go build main.go
go run .
打开浏览器：http://127.0.0.1:8080/main 进入首页

## ✨ 项目特色

- 🎯 **教学友好** - 源于尚硅谷 GoWeb 教程，代码结构清晰易懂
- 🗃️ **完善数据库** - 修复原 SQL 文件缺失表，优化数据结构
- 🔧 **功能完备** - 实现完整的电商核心功能
- 🐛 **问题修复** - 解决原项目中的常见问题和缺陷

## 🚀 功能特性

### 核心功能
- ✅ **用户管理** - 注册、登录、会话管理
- ✅ **图书管理** - 浏览、搜索、详情查看
- ✅ **购物车** - 添加商品、数量管理、价格计算
- ✅ **订单系统** - 下单、支付、订单历史
- ✅ **库存管理** - 图书库存监控和更新

### 技术特性
- 🏗️ **MVC 架构** - 清晰的分层设计
- 🔐 **安全认证** - Session 管理，安全验证
- 💾 **数据持久化** - MySQL 数据库存储
- 🎨 **前端界面** - 简洁美观的用户界面
